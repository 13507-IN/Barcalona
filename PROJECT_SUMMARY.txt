📊 PROJECT OVERVIEW
═══════════════════════════════════════════════════════════════════════════════

This is a PRODUCTION-READY full-stack web application that:
✅ Scrapes data from Wikipedia and sports news websites
✅ Serves data through a Flask REST API
✅ Displays information in a beautiful, responsive dashboard
✅ Includes proper error handling and data caching
✅ Works on desktop, tablet, and mobile devices
✅ Uses Barcelona's official colors (Blue & Garnet)
✅ Beginner-friendly with extensive comments

Total Lines of Code: ~2,500+ (well-documented)
Complexity: Intermediate (suitable for learning and production use)


📁 FILE STRUCTURE & DESCRIPTIONS
═══════════════════════════════════════════════════════════════════════════════

ROOT FILES:
───────────────────────────────────────────────────────────────────────

📄 app.py (156 lines)
   Purpose: Flask web application backend
   Contains: 6 API endpoints, error handlers, caching logic
   Key Features:
   - GET /api/club - Club information
   - GET /api/squad - Squad players
   - GET /api/news - News headlines
   - GET /api/matches - Upcoming & recent matches
   - GET /api/trophies - Trophy history
   - GET /api/all - All data combined
   Imports: Flask, CORS, scraper module
   Port: 5000

📄 scraper.py (463 lines)
   Purpose: Web scraping module
   Contains: 6 scraping functions + demo testing
   Key Features:
   - scrape_club_info() - Extracts club information from Wikipedia
   - scrape_squad() - Extracts player data
   - scrape_news() - Extracts news from BBC Sports
   - scrape_matches() - Creates fixture/result data
   - scrape_trophies() - Trophy history data
   - get_all_data() - Combines all data
   Fallback Data: Returns structured default data if scraping fails
   User-Agent: Browser headers to avoid blocking

📄 requirements.txt (20 lines)
   Purpose: Python package dependencies
   Packages:
   - Flask 2.3.3
   - Flask-CORS 4.0.0
   - requests 2.31.0
   - beautifulsoup4 4.12.2
   - lxml 4.9.3 (optional but recommended)
   Installation: pip install -r requirements.txt
